# 接口文档（全量清单）

本文档与仓库 **`app/api/**/route.ts`** 保持一致，用于快速检索路径、方法与鉴权要点。内置交互式文档站见 **`/{lang}/api-docs`**（扫描实现 + **`lib/utils/api-docs-endpoint-meta.ts`** 元数据）。**专项变更流水**见 **`docs/接口变更文档.md`\*\*。

**约定**：无前缀的 API 路径均指站点根路径下的 **`/api/...`**；鉴权以各 `route.ts` 实现为准，下表「鉴权要点」为摘要。

### profile 补充约定（2026-04-21）

- `PUT /api/profile` 修改邮箱时，需提交 `emailVerificationCode`（验证码由 `POST /api/auth/send-verification-code` 且 `type=change_email` 获取）。
- `socialLinks` 建议仅使用 `github`、`wechatQr`、`douyin`、`bilibili` 四个键；服务端读取兼容历史键名。
- 资料保存会同步更新 `users` 表中的 `email`、`avatar`、`displayName`，前端应以最新 `GET /api/profile` 返回为准刷新本地用户态。

## posts

| 路径                       | 方法                       | 实现文件                               | 简述                                | 鉴权要点                                     |
| -------------------------- | -------------------------- | -------------------------------------- | ----------------------------------- | -------------------------------------------- |
| `/api/posts`               | GET / POST                 | `app/api/posts/route.ts`               | 文章列表 / 创建                     | GET 多为公开；POST 需 **Bearer**（登录用户） |
| `/api/posts/{id}`          | GET / PUT / DELETE / PATCH | `app/api/posts/[id]/route.ts`          | 文章详情与更新                      | GET 视可见性；写操作需 **Bearer**            |
| `/api/posts/{id}/view`     | POST                       | `app/api/posts/[id]/view/route.ts`     | 浏览量 +1                           | 通常匿名可调用                               |
| `/api/posts/{id}/favorite` | GET / POST                 | `app/api/posts/[id]/favorite/route.ts` | 是否已收藏 / 切换收藏               | GET 匿名可查；POST 需 **Bearer**             |
| `/api/posts/{id}/like`     | GET / POST                 | `app/api/posts/[id]/like/route.ts`     | 点赞状态与总数 / 切换点赞           | GET 匿名可查；POST 需 **Bearer**             |
| `/api/posts/engagement`    | GET                        | `app/api/posts/engagement/route.ts`    | 批量查询多篇文章的点赞/收藏状态     | Query：`ids`；匿名占位；登录用户返回真实状态 |
| `/api/posts/slug/{slug}`   | GET / PATCH                | `app/api/posts/slug/[slug]/route.ts`   | 按 slug 获取详情 / 校验密码保护文章 | 公开（PATCH 通过 body.password 校验）        |
| `/api/posts/{id}/share`    | POST                       | `app/api/posts/[id]/share/route.ts`    | 记录「分享文章」打点（可选 Bearer） | 匿名可调用；登录则写入对应用户活动           |
