# 博客前端界面说明（按当前实现）

## 概述

本文档描述当前博客前端页面与管理页面的真实实现，包含路由、核心交互、依赖接口与常见调试入口。  
项目为多语言路由，以下页面路径均以 `/{lang}` 为前缀（如 `/zh-CN`、`/en-US`、`/ja-JP`）。

## 页面与路由

### 公开阅读页

- 文章列表：`/{lang}/blog`
- 文章详情：`/{lang}/blog/{slug}`
- 标签页：`/{lang}/tags`
- 分类页：`/{lang}/categories`

### 管理页（登录后）

- 文章管理：`/{lang}/blog/manage`
- 新建文章：`/{lang}/blog/manage/create`
- 编辑文章：`/{lang}/blog/manage/edit/{id}`

## 前端能力（当前）

### 列表页 `/{lang}/blog`

- 支持搜索、排序、分页。
- 支持通过查询参数按标签筛选：`tagId`。
- 列表会包含 `password` 可见性文章（通过 `includePasswordProtected=true`），并在卡片上显示锁态。
- 点赞/收藏按用户维度切换（登录后生效，未登录会跳转登录）。
- 点击阅读时前端会先调用 `POST /api/posts/{id}/view`；进入详情后 `GET /api/posts/slug/{slug}` 在通过可见性/密码校验时也会触发一次浏览计数。

### 密码保护文章

- 非作者访问密码文章时，前端先弹出密码输入框。
- 通过 `PATCH /api/posts/slug/{slug}` 校验密码，成功后跳转详情并附带 `?password=...`。

### 文章管理页

- 支持创建、编辑、删除、发布状态管理。
- 支持可见性（`public` / `private` / `password`）与标签、分类关联。
- 支持特色图上传（见 `docs/博客特色图与MinIO上传.md`）。

## 技术栈（当前）

- Next.js 15（App Router）
- React 19 + TypeScript（strict）
- HeroUI（`@heroui/*`）
- Tailwind CSS 4
- Lucide React

## 前端主要文件

- 列表页核心：`components/blog/blog-page-content.tsx`
- 文章卡片：`components/blog/post-card.tsx`
- 前端文章 API 客户端：`lib/api/posts.ts`
- 文章 Hook：`lib/hooks/usePosts.ts`

## 依赖接口（前端视角）

- `GET /api/posts`：文章列表（分页、筛选、排序）
- `GET /api/posts/slug/{slug}`：按 slug 获取详情
- `POST /api/posts/{id}/view`：浏览计数
- `POST /api/posts/{id}/like`：点赞切换
- `POST /api/posts/{id}/favorite`：收藏切换
- `GET /api/posts/engagement`：批量获取点赞/收藏状态
- `PATCH /api/posts/slug/{slug}`：密码文章校验

接口全量清单请以 `docs/接口文档.md` 为准。

## 本地运行

```bash
pnpm install
pnpm dev
```

常用地址示例：

- `http://localhost:3000/zh-CN/blog`
- `http://localhost:3000/zh-CN/blog/manage`
